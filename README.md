# Parvus [![Build Status](https://travis-ci.org/neanias/parvus.svg)](https://travis-ci.org/neanias/parvus)

Yet another URL shortening gem

## Installation

    $ gem install parvus

## Usage

For usage instructions

    $ parvus (help)

To shorten urls

    $ parvus shorten (-s SERVICE) URL
